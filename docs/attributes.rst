The options for the enumerated attributes are:

* ``bias_start`` and ``bias_end``:

  * ``horizontal``
  * ``vertical``

* ``label_position``:

  * ``bottom``
  * ``bottom_left``
  * ``bottom_right``
  * ``center``
  * ``top``
  * ``top_left``
  * ``top_right``

* ``text_orientation``:

  * ``horizontal``
  * ``vertical``
